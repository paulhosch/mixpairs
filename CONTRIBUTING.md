# Contributing

Thanks for your interest in improving `mixpairs`.

## Development setup

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -e ".[dev]"
```

## Run tests

```bash
.venv/bin/pytest
```

## Pull requests

- Open a PR against `main`.
- Keep changes focused and minimal.
- Ensure tests pass before requesting review.
- Prefer clear, meaningful naming and avoid redundant code.
